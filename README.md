# Nirby

Plateforme pour garder, organiser et partager des lieux/activités.

## Stack

- Frontend: React + Vite (TypeScript)
- Backend: Express (TypeScript)
- DB: PostgreSQL + PostGIS
- Orchestration: Docker Compose
- Pkg manager: pnpm
